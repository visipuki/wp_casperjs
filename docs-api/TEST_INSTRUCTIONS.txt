Инструкции:
1. изначально открывается корневая страница сайта (/)
2. из нее можно начать выполнять неавторизованные действия: 	['listPostsByArchive','listPostsByFrontPage','listPostsByCategory','listPostsByTag','search']
Все эти действия приведут к какому-то посту (в случае удачного поиска).
3. Из какого-нибудь поста можно вернуться обратно на корневую страницу командой openHome, или можно оставить комментарий командой comment (эта команда не изменит url и поэтому можно вызывать еще ее неоднократно или же командой openHome вернуться на корневую)
4. Так же из корневой можно ввести 'login username passw' (аргументы не обязательны, но по умолчанию стоят невалидные значения), тогда в случае успеха откроется меню админа, в случае неуспеха меню авторизации (надо ввести openHost, которая просто делает casper.open(host), где host не меняется с самого начала)
5. Из меню админа можно разлогиниться ('logoutUpBar' или 'logoutMetaLeft'), тогда откроется меню авторизации (надо ввести openHost, чтобы оказаться в корне)
6. Так же из меню админа можно открыть любой раздел для администрирования: 'openTags','openPages','openLibrary','openCategories','openComments'
7. Из каждого раздела администрирования можно начать выполнять соответствующие действия по созданию, редактированию, удалению, bulk-действию (все конкретные команды выписаны в main.js или можно посмотреть в модулях), а так же можно сделать сортировку ('anySort').
8. После любой команды из пункта 7 необходимо сделать openDashboard, чтобы снова оказаться в меню админа.
9. Дополнительно обращаю внимание, что для создания новых постов необходимо предварительно собрать все теги, то есть из меню админа порядок команд следующий:
openTags, getAllTags, openPosts
Далее с этого момента можно выполнять любую команду для постов, требующую знания всех тегов. Для остальных команд над постами (сортировка по столбцам anySort и удаления поста deletePosts можно из пункта 6 вызвать openPosts)
