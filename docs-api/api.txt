List of modules:
1. unauth-everything.js:
	Модуль со всеми шестью действиями неавторизованного пользователя:
		1.1. listPostsByTag(casper) - открывает произвольный пост по произвольному тегу
		1.2. listPostsByCategory(casper) - открывает произвольный пост по произвольной категории
		1.3. listPostsByFrontPage(casper) - открывает произвольный пост по произвольной странице
		1.4. listPostsByArchive(casper) - открывает произвольный пост по произвольной дате архива
		1.5. search(casper) - делает поисковый запрос и открывает произвольный среди найденных пост
		1.6. comment(casper) - комментирует текущий пост или комментарий к текущему посту
		
2. comment-everything.js:
	Модуль со всеми семью действиями во всех пяти категориях для администрирования комментариев:
		2.1. countComments - возвращает количество комментариев на текущей открытой странице
		2.2. showComments - выводит на экран все комментарии с текущей страницы
		2.3. spamComments - помечает случайный комментарий с текущей страницы спамом
		2.4. aproveOrUnaproveComments - помечает случайный комментарий с текущей страницы проверенным/непроверенным
		2.5. quickEditComments - проводит быстрое редактирование случайного комментария с текущей страницы
		2.6. editComments - проводит полное редактирование случайного комментария с текущей страницы
		2.7. replyComments - создает комментарий-ответ на случайный комментарий с текущей страницы
		2.8. deleteComments - удаляет/удаляет навсегда (в случае разделов Spam и Trash) случайный комментарий с текущей страницы
		2.9. notSpamComments - убирает метку спам со случайного комментария с текущей страницы
		2.10. bulkComments - проводит случайное bulk-действие случайного комментария с текущей страницы
		2.11. secAll - проводит случайное действие из вкладки All
		2.12. secPending - проводит случайное действие из вкладки Pending
		2.13. secApproved - проводит случайное действие из вкладки Approved
		2.14. secSpam - проводит случайное действие из вкладки Spam
		2.15. secTrash - проводит случайное действие из вкладки Trash
		2.16. performRandomCommentAction - выбирает случайную вкладку и переходит на нее и вызывает соответствующую функцию(secAll, secSpam, etc)
		2.17. openSecAll - переходит на вкладку  и выбирает случайную на ней страницу
		2.18. openSecPending - переходит на вкладку Pending и выбирает случайную на ней страницу
		2.19. openSecApproved - переходит на вкладку Approved и выбирает случайную на ней страницу
		2.20. openSecSpam - переходит на вкладку Spam и выбирает случайную на ней страницу
		2.21. openSecTrash - переходит на вкладку Trash и выбирает случайную на ней страницу
		2.22. openComments - из меню администрирования переходит к администрированию комментариев
		
3. page-everything.js:
	Модуль со всеми пятью (create, delete, bulk, full-edit, quick-edit) действиями для администрирования страниц:
		3.1. countPages(casper) - возвращает количество страниц-постов на текущей открытой странице
		3.2. showPages(casper, numOfPages) - выводит на экран все страницы-посты с текущей страницы
		3.3. bulkEditPages(casper, num) - выполняет bulk-редактирование уже выделенных элементов
		3.4. createPages(casper) - создает новую пост-страницу
		3.5. deletePages(casper) - удаляет пост-страницу
		3.6. fullEditPages(casper) - производит полное редактирование
		3.7. quickEditPages(casper)- производит быстрое редактирование
		3.8. bulkActionPages(casper)- выполняет bulk-действие
		3.9. openPages(casper) - из меню администрирования переходит к администрированию пост-страниц
		
4. auth-everything.js:
	Модуль с тремя действиями (залогиниться, разлогиниться, отсортировать таблицу по столбцу):
		4.1. login(casper) - берет из config.js username:password, проверяет залогинен ли пользователь сейчас, если нет, то логиниться (успех зависит от данных в config.js)
		4.2. logoutUpBar(casper) - проверяет залогинен ли пользователь, если да, то разлогинивается нажатием ссылки Log out в верхней меню
		4.3. logoutMetaLeft(casper) - проверяет залогинен ли пользователь, если да, то разлогиниваетсяся нажатием ссылки Log out в левом меню
		4.4. anySort(casper) - берет ссылку на таблицу из config.js, сортирует по случайному возможному столбцу
	
5. gen_text.js:
	Модуль для генерации текста (весь текст берется из переменной text), содержит три функции:
		5.1. getRandomTextLetters(min, max) - генерирует текст, длина которого в пределах [min, max] символов
		5.2. getRandomTextWords(min, max) - генерирует текст, длина которого в пределах [min, max] слов
		5.3. getRandomTextWordsTags(casper, textSel, min, max) - заполняет форму по селектору textSel текстом, длина которого в пределах [min, max] слов со вставленными в него допустимыми html-тегами, теги сбалансированы и возможно вложенные
		
6. categories-everything.js:
	Модуль по администрированию категорий:
		6.1. createCategories(casper) - создает новую категорию
		6.2. deleteCategories(casper) - удаляет случайную категорию
		6.3. editCategories(casper) - редактирует случайную категорию
		6.4. quickEditCategories(casper) - производит быстрое редактирование случайной категории
		6.5. bulkAction(casper) - производит bulk-действие над случайной категорией
		6.6. openCategories(casper) - открывает раздел с категориями
		
7. tags-everything.js:
	Модуль по администрированию тегов:
		7.1. createTags(casper) - создает новый тег
		7.2. deleteTags(casper) - удаляет случайный тег
		7.3. editTags(casper) - редактирует случайный тег
		7.4. quickEditTags(casper) - производит быстрое редактирование случайного тега
		7.5. bulkAction(casper) - производит bulk-действие над случайным тегом
		7.6. openTags(casper) - открывает раздел с тегами
		
8. post-everything.js:
	Модуль по администрированию постов:
		8.1. getAllTags(casper, allTags) - со странцы с тегами просматривает все теги и записывает их названия в allTags
		8.2. createPosts(casper, allTags) - создает новый пост, используя известные теги allTags
		8.3. fullEditPosts(casper, allTags) - производит полное редактирование поста
		8.4. quickEditPosts(casper, allTags) - производит быстрое редактирование поста
		8.5. bulkActionPosts(casper, allTags) - производит bulk-действие над постом
		8.6. deletePosts(casper) - удаляет пост
		8.7. openPosts(casper) - открывает страницу с постами из меню администратора
		
9. media-everything.js:
	Модуль по администрированию медиафайлов:
		9.1. uploadBrowser(casper) - закачивает медиафайл из перечня через стандартное окно для браузеров (в данном случае проходит POST-запрос)
		9.2. uploadMultiFile(casper) - закачивает медиафайл из перечня через окно для мультивыбора файлов с функцией drop (в данном случае проходит ajax-запрос)
		9.3. editMedia(casper) - производит редактирование описания медиафайла
		9.4. bulkMedia(casper) - производит bulk-действие над медиафайлом
		9.5. deleteMedia(casper) - удаляет медиафайл
		9.6. openLibrary(casper) - открывает страницу с медиафайлами из меню администратора
		
10. config.js:
	Модуль с конфигурациями:
		10.1. getUrl(path) - возвращает url корневой страницы блога
		10.2. setUrl(value) - устанавливает значение value для url корневой страницы блога
		10.3. getUsername() - возвращает имя пользователя (админа) по умолчанию
		10.4. setUsername(value) - меняет значение имени пользователя по умолчанию
		10.5 getMediaPath() - возвращает путь к каталогу с медиафайлами
		10.6. getPassword() - возвращает пароль для администратора по умолчанию
		10.7. setPassword(value) - меняет пароль для администратора по умолчанию
		10.8. getSpecificPostUrl() - возвращает url конкретного поста по умолчанию
		10.9. setSpecificPostUrl(value) - меняет url конкретного поста по умолчанию
		10.10. getSpecificTableUrl() - возвращает url на таблицу по умолчанию
		10.11. setSpecificTableUrl(value) - меняет url таблицы по умолчанию
		
11. common.js:
	Модуль общих функций:
		11.1. getRandomInt(min, max) - возвращает случайное число из диапозона [min, max)
		11.2. getRandomName() - возвращает случайное имя из массива list_of_names
		11.3. getRandomWebsite() - возвращает случайный url вебсайт из массива list_of_websites
		11.4. getRandomMedia() - возвращает случайное название медиафайла из массива list_of_media_files
		11.5. getRandomEmail() - возвращает случайный email из массива list_of_emails
		11.6. getRandomWord() - возвращает случаное слово из массива voc_list_new
		11.7. refresh(casper) - обновляет текущую страницу, удаляя все GET-параметры
		11.8. querySelectorAllinnerHTML(casper, sel) - по селектору sel берет список элементов из DOM-объекта страницы и возвращает массив, составленный из их innerHTML
		11.9. getShowOpenPosts(casper) - при условии, что текущая страница содержит список постов, выбирает один произвольный и открывает его
		11.10. checkLoggedIn(casper, mode) - с корневой страницы блога проверяет залогинена ли сессия, mode позволяет указать реакцию (0 - выйти, если разлогинен, 2 - выйти, если залогинен, 1 - ничего не делать, стоит по умолчанию), вернет true если залогинен и false в противном случае
		11.11. changePage(casper) - с web-страницы с таблицей элементов (посты, комменты и тд) выбирает произвольную страницу таблицы и переходит на нее
		11.12. fillSelect(casper, selector) - по селектору selector выбирает произвльное значение для html-тега <option>
		11.13. makeCheck(casper, selector) - по селектору selector кликает на <input type="checkbox">
		11.14. openHost(casper, host) - аналогично casper.open(host)
